# Lab 01 – Setting up a Python-Based ML Workbench

## Course Information

**Course Code:** CSE3011
**Course Title:** Machine Learning Techniques

---

## Lab Title

Setting up a Python-Based Machine Learning Workbench

---

## Aim

To set up a Python-based machine learning environment using Google Colab and verify the successful installation and functioning of essential machine learning libraries such as NumPy, Pandas, Matplotlib, and Scikit-learn.

---

## Objectives

1. Verify the availability of a Python-based ML environment.
2. Import and test essential machine learning libraries.
3. Explore real-world datasets using Python.
4. Generate basic visualizations to verify plotting capabilities.
5. Confirm readiness for subsequent machine learning laboratory exercises.

---

* NumPy

* Scikit-learn
* GitHub
* GitHub Codespaces

## Results
The Python machine learning environment was successfully configured using Google Colab. Essential machine learning libraries including NumPy, Pandas, Matplotlib, and Scikit-Learn were imported and verified successfully.

Dataset A (Microsoft Cloud Monitoring Dataset) was loaded, explored, and visualized successfully. Dataset B (OpenStack Failure Dataset) was also loaded and inspected successfully. Class distributions and dataset characteristics were examined using basic data exploration techniques.

The generated visualizations confirmed that the plotting environment was functioning correctly and the datasets were ready for further machine learning experiments.


---

## Conclusion

The objective of setting up a Python-based machine learning workbench was successfully achieved. Google Colab provided a suitable environment for executing machine learning workflows. All required libraries were verified, and both cloud infrastructure datasets were loaded and analyzed successfully. The environment is now ready for subsequent laboratory exercises involving machine learning model development and evaluation.
