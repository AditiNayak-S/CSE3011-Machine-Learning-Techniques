## Project Overview

This repository contains laboratory work, dataset exploration, machine learning implementations, and comparative analyses conducted as part of the CSE3011 Machine Learning Techniques course.

The project focuses on understanding, monitoring, analyzing, and predicting failures in cloud computing environments using real-world datasets from Microsoft Cloud Monitoring systems and OpenStack fault-injection experiments.

---

## Datasets

### Dataset A – Microsoft Cloud Monitoring Dataset

**Type:** Time-Series Anomaly Detection Dataset

**Characteristics:**

* 60 cloud monitoring time-series datasets
* Expert-labelled anomalies
* Real Microsoft production telemetry
* Monitoring metrics such as crash rates, API latency, ingress rates, and service availability

**Applications:**

* Time-Series Analysis
* Anomaly Detection
* Feature Engineering
* Cloud Monitoring Analytics

---

### Dataset B – OpenStack Failure Dataset

**Type:** Multi-Class Failure Classification Dataset

**Characteristics:**

* Fault-injection experiments on OpenStack
* DEPL, NET and STO workloads
* Multi-class failure labels
* Event-based feature matrices

**Applications:**

* Failure Diagnosis
* Failure Classification
* Ensemble Learning
* Cloud Reliability Analysis
